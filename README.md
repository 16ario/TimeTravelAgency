# TimeTravel Agency — Webapp interactive

TimeTravel Agency est une webapp pédagogique mettant en scène une entreprise fictive de voyage par trou de ver et tunnel spatio-temporel.

L’application permet aux visiteurs de découvrir trois époques d’arrivée, de consulter les détails de chaque transfert, de discuter avec un agent conversationnel simulé et de planifier une demande de réservation.

## Concept

TimeTravel Agency utilise des tunnels spatio-temporels stabilisés pour transporter ses clients vers des périodes historiques précises.

Les destinations proposées sont :

- Paris 1889
- Le Crétacé
- Florence 1504

Chaque transfert est présenté avec :

- une période d’arrivée ;
- un type de tunnel ;
- un niveau de risque ;
- une durée recommandée ;
- un prix estimé ;
- des activités ;
- un protocole de sécurité.

## Technologies utilisées

- HTML
- CSS
- JavaScript vanilla

Aucun framework n’est nécessaire pour cette première version.

## Fonctionnalités

- Page d’accueil immersive
- Vidéo de fond dans la hero section
- Présentation de l’entreprise
- Galerie de destinations
- Cards interactives
- Détails dynamiques par destination
- Vidéos intégrées par époque
- Agent conversationnel simulé
- FAQ
- Formulaire de réservation
- Validation automatique du formulaire
- Design responsive

## IA utilisée

Le projet a été conçu avec l’aide de ChatGPT pour :

- structurer l’architecture de la webapp ;
- générer les fichiers HTML, CSS et JavaScript ;
- adapter les textes au concept de voyage par trou de ver ;
- créer la logique du chatbot local ;
- améliorer la présentation pédagogique.
